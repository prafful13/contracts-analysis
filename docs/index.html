contracts-analysis

Local options screener — income plays ranked by annual yield, directional plays by composite score

GitHub ↗ Portfolio

What it does

Income Screener

Finds cash-secured puts and covered calls to sell. Ranked by annualised return on collateral — the Annual% column renders as a progress bar so the best contracts are instantly scannable.

Buy Screener

Surfaces high-delta directional plays (bullish calls, bearish puts). Ranked by a composite score: delta×100 + volume÷100 + OI÷1000.

Live Data

Fetches real option chains from Yahoo Finance via yfinance. Greeks (delta, gamma, theta, vega) computed analytically via py_vollib (Black-Scholes).

Configurable Without Rebuilding

Tickers, DTE range, delta range, and liquidity thresholds all live in config.yaml. In k3s, it's mounted as a ConfigMap — change and restart, no image rebuild.

Quick Start

# local dev
uv sync --all-groups
uv run inv run          # → http://localhost:8501

# k3s deployment (Rancher Desktop)
uv run inv bootstrap      # namespace via k3s-dev
uv run inv docker-build   # docker build
uv run inv k8s-apply      # → http://localhost:30502
